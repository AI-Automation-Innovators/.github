## Get started (fast)
1. **Clone a template:** pick a repo ‚Üí `Use this template` or **Fork**.
2. **Import a flow:** in n8n ‚Üí *Import from file* ‚Üí choose a JSON from `/n8n/json/`.
3. **Configure env:** add keys (Supabase, OpenAI/Claude, etc.) as `.env` or secrets.
4. **Run & iterate:** test locally, then open a PR with improvements.
